# FreshMart Dashboard
[Live Demo]https://saharacodie.github.io/FreshMart-Supermarket---Sales-Inventory-Analytics-/

## Overview

FreshMart Dashboard is a simple web-based analytics application built using HTML, CSS, and JavaScript. It provides a basic login system and displays key business insights such as total sales, profit, and stock availability using charts.

## Features

* User login authentication (static credentials)
* KPI cards for quick insights
* Bar chart for sales visualization
* Pie chart for category distribution
* Responsive and clean UI design
* Logout functionality

## Technologies Used

* HTML
* CSS
* JavaScript
* Chart.js (for data visualization)

## Project Structure

* Login Page
  A simple login interface with background image and glass effect UI.

* Dashboard Page
  Displays:

  * Total Sales
  * Profit
  * Days Out of Stock
  * Charts (Bar and Pie)
